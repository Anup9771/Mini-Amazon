# Mini Amazon

Mini Amazon is a simple e-commerce web application inspired by Amazon.  
This project is made for learning and practice purposes.

---

## Features

- User Login & Signup  
- Product Listing  
- Add to Cart  
- Remove from Cart  
- Order Summary  

---

## Tech Stack

- HTML, CSS, JavaScript  
- Backend: Node.js / Python *(as used)*  
- Database: MongoDB / MySQL *(as used)*
